# 放置点击游戏

一个使用 React 开发的放置类点击游戏，玩家通过击败怪物获取装备和技能，升级装备增强战斗力，收集图鉴提升属性！

## 功能特性

- ⚔️ **装备系统**：6个装备位置（武器、头盔、护甲、护腿、靴子、饰品），每个装备都有攻击力和防御力属性
- 🗡️ **战斗系统**：与怪物战斗，击败小怪积攒Boss进度、挑战Boss
- 📖 **图鉴系统**：收集怪物图鉴，永久增加角色属性
- 🎯 **关卡系统**：关卡越高，怪物越强，掉落越好
- 📊 **角色等级**：击败怪物获取经验自动升级，等级提升提供基础攻击、防御、生命等增益
- 🧠 **多维属性**：装备和怪物均拥有攻击、防御、生命、暴击率与暴击伤害等多维度属性
- ⚡ **技能系统**：击败怪物有概率掉落技能，可装备最多3个技能提升属性
- 💾 **自动保存**：游戏进度自动保存到本地存储

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 游戏玩法

1. **战斗怪物**：击败小怪积累Boss进度，进度条满后自动召唤Boss
2. **Boss挑战**：Boss必定掉落史诗及以上品质装备，击败后自动解锁下一关
3. **装备养成**：装备拥有攻击、防御、生命、暴击率与暴击伤害等属性，可通过背包自由搭配
4. **图鉴收集**：首次击败某个怪物可解锁图鉴，获得永久属性加成
5. **关卡选择**：可重复挑战已通关关卡进行刷装
6. **隐藏密码**：连续按下三次 `k` 键即可永久增加5000攻击力与5000防御力（仅一次）
7. **升级养成**：击败怪物可获取经验，经验溢出自动升级并回满血量；等级越高，基础属性越强

## 技术栈

- React 18

## 怪物系统

游戏包含8种怪物，每种怪物都有不同的属性：

1. **哥布林** 👹 - 基础怪物
2. **兽人** 👺
3. **骷髅** 💀
4. **巨龙** 🐉
5. **恶魔** 😈
6. **泰坦** 👑
7. **凤凰** 🔥
8. **神祇** ✨

击败怪物有概率掉落装备和技能，Boss必定掉落高品质装备。

## 装备属性

每个装备位置提供不同的属性加成：
- **武器** ⚔️：主要提供攻击力
- **头盔** 🪖：提供少量攻击和防御
- **护甲** 🛡️：主要提供防御力
- **护腿** 🦵：提供少量攻击和防御
- **靴子** 👢：提供少量攻击和防御
- **饰品** 💍：平衡提供攻击和防御
